How the data flows

Each sector card opens the same dashboard shell, pointed at that sector via dashboard.html?sector=<id>. The dashboard fetches the sector's clean CSV, builds the data model in the browser, and renders the charts — no server needed. The CSV location is set once in assets/js/sectors.js.
