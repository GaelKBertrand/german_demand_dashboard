Duplicate filter (enabled by default): Every job posting is counted as its own post. This filter only removes rows that carry the exact same Job ID more than once — a scraping artifact, not a real second posting. In the current datasets every Job ID is unique, so no postings are removed and the filter has no effect on the numbers.
